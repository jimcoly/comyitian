What other extensions we have made?

IE Tab Classic
IE Tab lets you use Internet Explorer inside Chrome. Some websites are still not compatible with standard browsers, so if you need access to those websites, this extension will assist you.
FlashBlock
This extension automatically blocks all the flash contents on any webpage. Each flash element is being replaced with a placeholder that allows you to load only selected elements on a given page. You can also manage a whitelist of allowed websites via a configuration panel.
Chrome Flags
Chrome Flags displays a country flag indicating about the physical location of the websites you're visiting. The extension also provides access to detailed information regarding the website's servers, owners, traffic and trustworthiness.
